Welcome!

Welcome to fontawesome-bulder. This is a free project that will allow you to render the fontawesome icons using PHP.

The structure of the documentation is based on the official fontawesome docs for you to be easier to read how to use the library to render the icons.

Installation

You can install fontawesome-builder into your project using Composer. Open your terminal navigate to your project's folder and run the following:

composer require ditoskas/fontawesome-builder
